
# Project Presentation

This folder contains the presentation slides summarizing the key findings from the Market Analysis Dashboard project.

### The presentation highlights:

- Project overview and business problem
- Customer demographic analysis by age, gender, income group, and country
- Sales performance and regional distribution
- Identification of top and underperforming products
- Time-series analysis of yearly, quarterly, and monthly sales trends
- Product-level drill-through insights
- Strategic recommendations based on dashboard findings
