
# Power BI Dashboard

This folder contains the Power BI dashboard developed for the Market Analysis project.

The dashboard provides an interactive view of sales performance, customer demographics, product performance, and time-based trends.

## Key dashboard features include:

• KPI indicators for Total Sales, Total Orders, Total Customers, and Average Monthly Sales
• Customer demographic analysis by Age Group, Gender, Income Group, and Country
• Product performance analysis identifying top and bottom performing products
• Sales distribution across key Latin American markets
• Time-series analysis including Yearly, Quarterly, and Monthly sales trends
• Drill-through functionality for detailed product-level insights

The dashboard enables decision-makers to explore performance trends and identify growth opportunities.
